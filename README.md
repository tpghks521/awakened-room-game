# 깨어난 방 (Awakened Room) - 웹 버전

3D 공포 퍼즐 게임 - Three.js로 제작

## 현재 구현된 기능 ✅

### Phase 1: 기본 환경
- ✅ 3D 방 생성 (벽, 바닥, 천장)
- ✅ 1인칭 시점 카메라
- ✅ WASD 이동 + 마우스 시점 조작
- ✅ Shift 달리기

### Phase 2: 조명
- ✅ 어두운 분위기 (공포 게임용)
- ✅ 포인트 라이트 (중앙 조명)
- ✅ 깜빡이는 조명 효과
- ✅ 그림자 효과

### Phase 3: 퍼즐 시스템
- ✅ 3개의 문 생성
- ✅ 문 상호작용 (E키)
- ✅ 정답/오답 문 구분 (문 2가 정답)
- ✅ 정답 시 승리, 오답 시 게임 오버

### Phase 4: UI
- ✅ 게임 시작 화면
- ✅ 조작법 안내
- ✅ 십자선 (크로스헤어)
- ✅ 힌트 표시
- ✅ 상호작용 프롬프트
- ✅ 게임 오버 화면
- ✅ 승리 화면

## 아직 구현 안 된 기능 ⏳

- ⏳ 배경 음악 및 효과음
- ⏳ 괴물 AI 및 추격 시스템
- ⏳ 복도/추가 방
- ⏳ 플레이어가 뒤돌 때 조명 꺼지는 효과
- ⏳ 더 복잡한 퍼즐

## 게임 실행 방법

### 방법 1: 로컬 서버 사용 (권장)

1. **Python이 설치되어 있다면:**
   ```bash
   cd C:\Users\ha_du\awakened-room-game
   python -m http.server 8000
   ```
   그리고 브라우저에서 `http://localhost:8000` 열기

2. **Node.js가 설치되어 있다면:**
   ```bash
   cd C:\Users\ha_du\awakened-room-game
   npx http-server
   ```
   그리고 브라우저에서 표시된 주소 열기

### 방법 2: 브라우저에서 직접 열기

파일 탐색기에서 `index.html`을 더블클릭하면 브라우저에서 열립니다.

**주의:** 일부 브라우저에서는 보안 정책상 로컬 파일에서 일부 기능이 제한될 수 있습니다. 가능하면 방법 1 (로컬 서버)를 사용하세요.

## 조작법

### 💻 PC
- **WASD** 또는 **방향키** - 이동
- **마우스** - 둘러보기 (화면 클릭해서 마우스 잠금)
- **E** - 문 열기 (가까이 가면 프롬프트 표시)
- **Shift** - 달리기

### 📱 모바일
- **왼쪽 조이스틱** - 이동
- **화면 스와이프** - 카메라 회전
- **E 버튼** - 문 열기 (문 근처에서 자동으로 나타남)
- **달리기 버튼** - 빠르게 이동

**모바일 팁:**
- 조이스틱으로 방향 조절
- 화면 중앙을 터치하고 드래그하면 둘러볼 수 있습니다
- 괴물한테 쫓길 때는 달리기 버튼을 누른 채로 앞으로!

## 게임 목표

1. 어두운 방에서 깨어납니다
2. 앞쪽에 3개의 문이 있습니다
3. 올바른 문을 선택하면 탈출 성공!
4. 잘못된 문을 선택하면... 게임 오버

**힌트:** 각 문 위에 작은 색깔 표시가 있습니다
- 🟢 초록색 = 정답 문
- 🔴 빨간색 = 오답 문

## 문제 해결

### 게임이 로드되지 않을 때
- 브라우저 콘솔(F12)을 열어서 에러 확인
- 인터넷 연결 확인 (Three.js CDN 로드를 위해 필요)

### 마우스 조작이 안 될 때
- 화면을 한 번 클릭해서 포인터 잠금 활성화
- ESC를 누르면 포인터 잠금 해제

### 화면이 너무 어두울 때
- `game.js`에서 조명 설정 조절 가능
- `ambientLight` 강도를 높이면 밝아집니다
